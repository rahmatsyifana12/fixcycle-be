Frontend repository: https://github.com/leonardo-lim/fixcycle

Running at port 5000

### Technologies
* JavaScript
* Node.js
* Express.js
* PostgreSQL

### Getting Started

#### Install dependencies
```
npm install
```

#### Initialize your env
```
cp .env.example .env
```

#### Generate token secret
```
npm run jwt:generate
```

#### Seed data into database
```
npm run seed
```

#### Start the project
```
npm run start
```
